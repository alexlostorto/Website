Sparx Maths Extension

An extension created to increase the learning efficiency of students learning via SparxMaths.

Features

  • 3 pre-defined colour themes top choose from
  • Stores the answer with the bookwork code
  • Highlights stored answer when there is a bookwork check

How to Install

Firstly, you need to click this link to download the .zip file.

Next, extract the files from the .zip file and move them to a separate folder.
Make sure this folder is in a location you can access.

Now go to Chrome Extensions and enable 'Developer mode' in the top right corner of the screen.

Click on 'load unpacked' at the top left corner of the screen and navigate to the folder where you saved the extension.

Pin the SparxMaths Extension to access the colour theme options and make sure to reload SparxMaths before using the extension.
